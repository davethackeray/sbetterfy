Set Up Your Google AI API Key

1

Create a Google AI Studio Account

You'll need a Google account to access Google AI Studio.

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Accept the terms of service
2

Get Your API Key

Now you'll need to generate an API key:

  1. In Google AI Studio, click on "Get API key" in the top right
  2. If you don't see this option, click on your profile picture and select "API keys"
  3. Click "Create API key" and give it a name (e.g., "Spotify Recommendations")
  4. Your API key will be displayed - copy it
  5. Keep this key secure and don't share it with others
